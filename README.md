**목적:** 공통 과제를 함께 수행하며 웹 개발의 전체 흐름을 빠르게 익히고 협업에 적응하기

**결과물:** 기획부터 배포까지 완료된 웹 서비스와 관련 문서 일체

## 기획안

> 프로젝트 주제, 목적, 핵심 기능, 예상 사용자, 팀원별 역할 등 정리

- **주제:** 콘서트 티켓팅 연습, 콘서트 정보 제공, 좌석 리뷰 기능을 통합한 웹 기반 콘서트 정보 플랫폼
- **목적:** 인기 콘서트 티켓팅 과정에서 발생하는 빠른 좌석 선택과 좌석 정보 부족 문제를 해결하고, 사용자가 사전에 티켓팅을 연습하며 공연 정보를 한곳에서 확인할 수 있도록 지원한다.
- **핵심 기능:**
  - **티켓팅 연습:** 실제 티켓팅 상황과 유사한 대기열, 좌석 선택, 시간 제한 환경을 제공하여 사용자가 티켓팅 과정을 미리 경험할 수 있다.
  - **콘서트 정보 제공:** 공연 목록, 공연 일정, 장소, 출연진, 티켓 오픈 시간, 가격대 등 콘서트 관련 정보를 제공한다.
  - **좌석 리뷰:** 사용자가 공연장 좌석별 시야, 음향, 거리감, 만족도 등을 리뷰로 남기고 다른 사용자의 후기를 확인할 수 있다.
  - **좌석 정보 확인:** 공연장별 좌석 배치도와 좌석 등급, 가격, 리뷰 정보를 함께 제공하여 좌석 선택 전 판단을 돕는다.
- **예상 사용자:**
  - 콘서트 티켓팅을 처음 시도하거나 예매 과정이 익숙하지 않은 사용자
  - 인기 공연 예매 성공률을 높이기 위해 사전 연습이 필요한 팬
  - 공연 일정, 장소, 티켓 오픈 시간 등 콘서트 정보를 한곳에서 확인하고 싶은 사용자
  - 공연장 좌석별 후기를 공유하고 다른 사용자의 리뷰를 참고하려는 관람객

## 기능 명세서

### 1. 기능 명세 개요

본 프로젝트는 콘서트 예매를 준비하는 사용자가 실제 티켓팅과 유사한 환경에서 사전 연습을 하고, 공연 정보와 공연장 좌석 정보를 한곳에서 확인할 수 있도록 지원하는 웹 기반 콘서트 정보 플랫폼이다.

사용자는 공연 목록과 상세 정보를 확인하고, 티켓팅 연습 기능을 통해 대기열 진입, 좌석 선택, 시간 제한 등 실제 예매 과정과 비슷한 흐름을 경험할 수 있다. 또한 공연장 좌석별 리뷰를 확인하거나 직접 작성하여 좌석 선택 전 시야, 음향, 거리감 등에 대한 정보를 참고할 수 있다.

### 2. 주요 사용자 흐름

#### 사용자 흐름 1: 콘서트 정보 확인

1. 사용자가 메인 페이지에 접속한다.
2. 공연 목록을 확인한다.
3. 관심 있는 공연을 선택한다.
4. 공연 상세 페이지에서 일정, 장소, 출연진, 티켓 오픈 시간, 가격 정보를 확인한다.
5. 해당 공연장의 좌석 정보 또는 티켓팅 연습 기능으로 이동한다.

#### 사용자 흐름 2: 티켓팅 연습

1. 사용자가 티켓팅 연습 페이지에 접속한다.
2. 연습할 공연 또는 공연장을 선택한다.
3. 대기열 화면을 거쳐 예매 화면에 진입한다.
4. 제한 시간 안에 좌석을 선택한다.
5. 선택한 좌석 정보를 확인한다.
6. 예매 성공 또는 실패 결과를 확인한다.

#### 사용자 흐름 3: 좌석 리뷰 확인 및 작성

1. 사용자가 공연장 또는 공연 상세 페이지에서 좌석 배치도를 확인한다.
2. 특정 구역 또는 좌석을 선택한다.
3. 해당 좌석의 리뷰, 시야, 음향, 거리감, 만족도 정보를 확인한다.
4. 사용자는 직접 좌석 리뷰를 작성할 수 있다.
5. 작성된 리뷰는 다른 사용자도 확인할 수 있다.

##### [ ] 좌석 리뷰 이미지 업로드 기능

사용자는 좌석 리뷰 작성 시 실제 좌석 시야 사진을 함께 업로드할 수 있다.

**세부 기능**

- 이미지 파일 업로드
- 리뷰와 이미지 연결
- 좌석별 이미지 조회
- 이미지 미리보기 제공

**사용자 관점**

사용자는 텍스트 리뷰뿐 아니라 실제 시야 사진을 참고하여 좌석을 판단할 수 있다.

##### [ ] 좌석 리뷰 좋아요 기능

사용자는 도움이 된 좌석 리뷰에 좋아요를 누를 수 있다.

**세부 기능**

- 리뷰별 좋아요 버튼 제공
- 좋아요 수 표시
- 인기 리뷰 정렬
- 중복 좋아요 방지

**사용자 관점**

사용자는 유용한 리뷰를 쉽게 구분할 수 있다.

##### [ ] 좌석 리뷰 정렬 기능

사용자는 좌석 리뷰를 원하는 기준으로 정렬할 수 있다.

**세부 기능**

- 최신순 정렬
- 만족도 높은 순 정렬
- 만족도 낮은 순 정렬
- 좋아요 많은 순 정렬

**사용자 관점**

사용자는 자신에게 필요한 리뷰를 더 빠르게 확인할 수 있다.

##### [ ] 로그인 및 마이페이지 기능

사용자는 계정을 통해 자신의 활동 내역을 관리할 수 있다.

**세부 기능**

- 회원가입
- 로그인
- 로그아웃
- 내가 작성한 리뷰 조회
- 내가 연습한 티켓팅 기록 조회

**사용자 관점**

사용자는 자신의 리뷰와 연습 기록을 지속적으로 관리할 수 있다.

##### [ ] 티켓 오픈 알림 기능

사용자는 관심 공연의 티켓 오픈 시간을 알림으로 확인할 수 있다.

**세부 기능**

- 관심 공연 등록
- 티켓 오픈 시간 저장
- 오픈 전 알림 표시
- 관심 공연 목록 관리

**사용자 관점**

사용자는 티켓 오픈 시간을 놓치지 않고 예매를 준비할 수 있다.

##### [ ] 반응형 웹 UI 개선

사용자는 PC뿐만 아니라 모바일 환경에서도 서비스를 이용할 수 있다.

**세부 기능**

- 모바일 화면에 맞는 공연 목록 UI
- 모바일 좌석 배치도 표시 최적화
- 터치 기반 좌석 선택
- 작은 화면에서도 읽기 쉬운 리뷰 UI 제공

**사용자 관점**

사용자는 다양한 기기에서 콘서트 정보 확인과 티켓팅 연습을 할 수 있다.

### 5. 기능 우선순위

#### 1순위: 반드시 구현해야 하는 기능

- 공연 목록 조회
- 공연 상세 정보 조회
- 티켓팅 연습 시작
- 대기열 시뮬레이션
- 좌석 선택 연습
- 시간 제한
- 티켓팅 연습 결과 확인
- 공연장 좌석 배치도 조회
- 좌석 리뷰 조회
- 좌석 리뷰 작성

#### 2순위: 시간이 남으면 구현할 기능

- 공연 검색
- 공연 필터링
- 티켓팅 연습 난이도 설정
- 좌석 매진 시뮬레이션
- 좌석 리뷰 정렬
- 좌석 리뷰 좋아요

#### 3순위: 추가 확장 기능

- 로그인 및 마이페이지
- 티켓팅 연습 기록 저장
- 랭킹 기능
- 좌석 리뷰 이미지 업로드
- 티켓 오픈 알림
- 반응형 웹 UI 고도화

### 6. 핵심 기능별 기대 효과

| 기능 | 기대 효과 |
|---|---|
| 티켓팅 연습 | 사용자가 실제 예매 전에 좌석 선택 과정을 미리 경험할 수 있다. |
| 대기열 시뮬레이션 | 인기 공연 예매 상황의 긴장감과 흐름을 체험할 수 있다. |
| 콘서트 정보 제공 | 공연 일정, 장소, 가격, 티켓 오픈 시간을 한곳에서 확인할 수 있다. |
| 좌석 배치도 제공 | 공연장 구조와 좌석 등급을 미리 파악할 수 있다. |
| 좌석 리뷰 | 실제 관람객의 후기를 바탕으로 좌석 선택 판단을 도울 수 있다. |
| 리뷰 작성 | 사용자 간 좌석 정보를 공유하는 커뮤니티 기능을 형성할 수 있다. |

### 7. 최종 정리용 체크리스트

#### 필수 기능

- [ ] 공연 목록을 조회할 수 있다.
- [ ] 공연 상세 정보를 조회할 수 있다.
- [ ] 공연 일정, 장소, 출연진, 티켓 오픈 시간, 가격 정보를 확인할 수 있다.
- [ ] 티켓팅 연습을 시작할 수 있다.
- [ ] 대기열 화면을 체험할 수 있다.
- [ ] 제한 시간 안에 좌석을 선택할 수 있다.
- [ ] 좌석 선택 결과를 확인할 수 있다.
- [ ] 공연장별 좌석 배치도를 확인할 수 있다.
- [ ] 좌석 등급과 가격 정보를 확인할 수 있다.
- [ ] 좌석별 리뷰를 조회할 수 있다.
- [ ] 좌석 리뷰를 작성할 수 있다.
- [ ] 주요 페이지 간 이동이 가능하다.

#### 선택 기능

- [ ] 공연을 검색할 수 있다.
- [ ] 공연을 날짜, 장소, 가격 등으로 필터링할 수 있다.
- [ ] 티켓팅 연습 난이도를 설정할 수 있다.
- [ ] 좌석이 실시간으로 매진되는 상황을 시뮬레이션할 수 있다.
- [ ] 티켓팅 연습 기록을 저장할 수 있다.
- [ ] 티켓팅 연습 랭킹을 확인할 수 있다.
- [ ] 좌석 리뷰에 이미지를 첨부할 수 있다.
- [ ] 좌석 리뷰에 좋아요를 누를 수 있다.
- [ ] 좌석 리뷰를 최신순, 만족도순 등으로 정렬할 수 있다.
- [ ] 로그인 후 마이페이지에서 내 활동을 확인할 수 있다.
- [ ] 관심 공연의 티켓 오픈 알림을 받을 수 있다.
- [ ] 모바일 환경에서도 편리하게 사용할 수 있다.
